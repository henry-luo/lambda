Testing Instructions

  • Use Tab and Shift+Tab to navigate between focusable elements
  • Use Arrow keys to move the caret in text fields
  • Hold Shift + Arrow keys to extend selection
  • Use Ctrl/Cmd + A to select all text in a focused input
  • Use Home/End to move to line start/end
  • Click to focus, observe :focus styles
  • Tab to focus, observe :focus-visible outline (dashed orange)
