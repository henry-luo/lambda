Table Simple Test Cases

About This Test Suite

This collection contains simplified HTML table tests focused purely on table layout mechanics. All tests avoid text content and instead use colored div and span elements to isolate table layout behavior from text rendering concerns. Each test demonstrates a specific table feature or property.

001 - Basic Table Structure

A simple 2x2 table with colored div blocks demonstrating basic table cell layout.

002 - Border Collapse

Tests the border-collapse property with adjacent cell borders merging into a single border.

003 - Border Spacing

Demonstrates the border-spacing property with separate cell borders and custom spacing.

004 - Fixed Layout

Tests table-layout: fixed with predefined column widths that don't adjust to content.

005 - Colspan

Demonstrates the colspan attribute with cells spanning multiple columns using span elements.

006 - Rowspan

Tests the rowspan attribute with cells spanning multiple rows using sized div elements.

007 - Percentage Width

Tests percentage-based column widths within a fixed-width table container.

008 - Vertical Alignment

Demonstrates vertical-align property behavior (top, middle, bottom) in table cells.

009 - Empty Cells

Tests the empty-cells property showing the difference between 'show' and 'hide' values.
