### Code Block with Language

```python

    print("Hello, Lambda!")
    return 42

result = hello_world()
```

```javascript
function fibonacci(n) {
    if (n <= 1) return n;
    return fibonacci(n - 1) + fibonacci(n - 2);
}

console.log(fibonacci(10));
```
