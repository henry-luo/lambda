# Basic Math Test

## Simple Expressions
**Addition:** $a + b$

**Subtraction:** $x - y$

**Multiplication:** $p \cdot q$

**Division:** $\frac{a}{b}$

**Power:** $x^2$

**Square Root:** $\sqrt{x}$

**Basic Equality:** $a = b$

**Greek Letters:** $\alpha + \beta$

**Parentheses:** $(a + b)$

**End**
