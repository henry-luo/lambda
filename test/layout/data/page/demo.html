Lambda Script & Radiant

Pure functional data engine with browser-grade layout and rendering

End-to-end pipeline Transform · Validate · Layout · Render

Lambda Script parses documents into a rich typed data model, transforms and validates them with a pure functional language, then hands them to the Radiant engine for pixel-perfect HTML/CSS layout and multi-format rendering.

Open live demo transform ./lambda.exe convert data.json -t yaml validate ./lambda.exe validate data.json -s schema.ls render ./lambda.exe view test/html/index.html

Core workflows

Transform

Convert JSON, XML, CSV, Markdown, LaTeX and more via a single functional pipeline.

// detect input, emit YAML
./lambda.exe convert input.json -t yaml -o out.yaml

// scriptable transform
let doc  = input("invoice.pdf", 'auto')
let rows = extract_lines(doc)
format(rows, 'markdown')

Validate

Run schema-based validation across JSON, XML, and HTML with rich error reporting.

// load schema & validate
let schema = load_schema("doc_schema.ls", "Article")
let data   = input("article.xml", 'xml')
let result = validate(schema, data)

Layout

Exercise block, inline, flex, grid, and table algorithms with browser-compatible metrics.

// CSS layout test
./lambda.exe layout test/html/flex.html  --format text
./lambda.exe layout test/html/grid.html  --format text

Render

Drive Radiant to render HTML/CSS into interactive windows or static PDF, SVG, and PNG output.

// interactive viewer
./lambda.exe view test/html/index.html
// off-screen rendering
./lambda.exe render page.html -o page.pdf
