Test Coverage Summary

This test suite covers the following table layout features:

  • Basic Layout: Grid formation, width distribution, height calculation
  • Cell Spanning: Colspan and rowspan with various values and combinations
  • Vertical Alignment: All CSS vertical-align values in table context
  • Border Models: Separate and collapsed border handling
  • Width Algorithms: Auto, fixed, and percentage-based table sizing
  • Content Types: Text, nested tables, lists, and mixed content
  • Caption Handling: Different positioning and styling options
  • Edge Cases: Unusual structures and extreme values

Running Tests

To test the table layout implementation:

# Test individual files
./radiant.exe layout test/layout/data/basic/table_simple.html
./radiant.exe layout test/layout/data/basic/table_colspan_test.html
./radiant.exe layout test/layout/data/basic/table_complex_spans.html

# Run all table tests
for file in test/layout/data/basic/table_*.html; do
  echo "Testing: $file"
  ./radiant.exe layout "$file"
done
