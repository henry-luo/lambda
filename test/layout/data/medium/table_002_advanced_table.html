API Endpoints Reference

The following table lists all available API endpoints with their details:

Method Endpoint Description Status Rate Limit
GET /api/users Retrieve a list of all users with optional filtering parameters Active 100/hour
POST /api/users Create a new user account with validation Active 10/hour
PUT /api/users/{id} Update an existing user's information completely Beta 50/hour
PATCH /api/users/{id}/profile Partially update user profile data Deprecated 25/hour
DELETE /api/users/{id} Permanently delete a user account and all associated data Active 5/hour

Rate limits are enforced per authenticated user. Status indicators show the current stability of each endpoint.
