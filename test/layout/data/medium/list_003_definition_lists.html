API Definitions

The following definitions explain key terms used in the API:

Authentication
The process of verifying user credentials before granting access to protected resources.
Bearer Token
A security token included in the Authorization header of HTTP requests.
Format: Authorization: Bearer <token>
Rate Limiting
A mechanism that controls the number of requests a client can make within a specified time window.
Webhook
An HTTP callback that delivers real-time data to other applications when specific events occur.
Commonly used for notifications and data synchronization.

These definitions help establish a common vocabulary for API documentation.
