# Node.js Official Test Skip List

# Tests listed here are permanently skipped because they require features
# that are fundamentally unavailable in the Lambda JS engine.
#
# Format: test-filename.js  # reason
# Lines starting with # are comments. Blank lines are ignored.
#
# Categories:
#   [fork]       Uses child_process.fork() which spawns a Node binary
#   [icu]        Requires full ICU internationalization support
#   [sea]        Requires Node.js Single Executable Application support
#   [snapshot]   Requires V8 snapshot building API
#   [native]     Requires native C++ addons / node-gyp
#   [hang]       Known to hang (infinite loop, unimplemented event loop feature)
#   [crash]      Known to crash LambdaJS during default official-test runs
#   [timeout]    Known to hit the runner timeout or require unavailable signals
#   [spawn-node] Spawns 'node' binary with Node-specific CLI flags
#   [unimplemented] Requires a Node feature not implemented in LambdaJS yet
#
