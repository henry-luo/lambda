Contributing

Lambda Script welcomes contributions of all kinds — bug fixes, documentation improvements, new recipes, and feature proposals.

How to Contribute

  1. Fork the repository on GitHub
  2. Create a feature branch: git checkout -b my-feature
  3. Make your changes and add tests
  4. Run the test suite: make test
  5. Submit a pull request

Good First Contributions

  • Add a new recipe or example to the website
  • Improve documentation clarity or fix typos
  • Report bugs with reproducible test cases
  • Add format conversion test cases
